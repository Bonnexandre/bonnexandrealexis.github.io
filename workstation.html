Validation Checklist

This is the same kind of verification flow used in workstation provisioning and desktop support.

POST / BIOS Verification

  • System POST successful
  • CPU detected correctly
  • RAM detected correctly (capacity + speed)
  • Storage detected (SSD/HDD)
  • Fans detected + stable temps

Thermals & Airflow

  • Correct fan orientation (intake/exhaust)
  • Clean airflow path (no cable blocking)
  • Case panels close without pressure
  • Baseline idle temperature recorded

Readiness

  • UEFI boot mode enabled
  • Secure Boot supported
  • Ready for Windows 11 Pro installation
  • Prepared for virtualization labs
